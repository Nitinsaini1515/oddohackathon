# 🚀 AssetFlow - Enterprise Asset & Resource Management System

> **An ERP-based Asset & Resource Management System built for the Odoo Hackathon.**

AssetFlow helps organizations efficiently manage their physical assets and shared resources through a centralized platform. It replaces manual tracking using spreadsheets or paper logs with a smart, secure, and role-based digital system.

---

# 📖 Overview

Organizations often struggle to manage assets such as laptops, furniture, vehicles, projectors, and meeting rooms. AssetFlow provides a complete ERP solution to:

- Track assets throughout their lifecycle
- Prevent duplicate asset allocation
- Manage shared resource bookings
- Handle maintenance workflows
- Perform asset audits
- Generate reports and analytics
- Notify users of important events
- Improve operational efficiency
